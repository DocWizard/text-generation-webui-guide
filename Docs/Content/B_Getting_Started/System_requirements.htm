System requirements

Generally speaking, the more powerful your computer, the stronger models you can run locally on your machine. There are three main methods of running local LLMs, and each depends on the amount of system memory you have:

GPU inference: First, and most preferred, is to load the model entirely into your VRAM. For that, you will need a dedicated graphics card.

How to find out if you have a dedicated graphics card

If you're not sure if you have a dedicated or integrated graphics card, follow these steps:

  • Open the Start Menu
  • Search for Device Manager
  • Navigate to Display Adapters
  • If you see "Intel Integrated Graphics", you do not have a dedicated graphics card. If you see AMD or NVIDIA, you are good to go! If you see both, that means your computer has both dedicated and integrated graphics - which is also good for our purposes.
How to check how much VRAM you have

If you're not sure how much VRAM you have available, follow these steps:

  • Open the Start menu
  • Search for Task Manager and launch it
  • Navigate to Performance, then select your dedicated GPU - it will be either GPU0 or GPU1
  • Note the number located in the top right corner of the Dedicated GPU memory usage box. That number represents the amount of VRAM you have available.

The amount of VRAM you have available will determine the size of the model you can load. You should assume 8GB of VRAM to be the minimum, and 24GB to be optimal. Some people go as far as to install multiple graphics cards into their system to maximize the amount of VRAM available to them.

GPU + CPU inference: This method will use both your VRAM and RAM to load the model. Offloading parts of the model to your system memory will allow you to load larger models at a significant performance penalty.

How to check how much system memory you have

If you're not sure how much RAM (system memory) you have available, follow these steps:

  • Open theStart menu
  • Search for Task Manager and launch it
  • Navigate to Performance > Memory
  • Note the number located in the top right corner. That's the amount of RAM you have available on your computer.
