Download and first-time setup

Downloading text-generation-webui is best accomplished by cloning the repository. You will need to download and install GitHub Desktop on your system. The process is straightforward and should only take a minute.

Tip!
If you are already familiar with Git, you may use it instead to clone the repository by yourself.

Once you have GitHub Desktop installed on your system, go ahead and launch it.

Navigate to File > Clone repository. In the 'repository URL' field, paste the following URL:

https://github.com/oobabooga/text-generation-webui

Next, in the field below, select the path you wish to use for your local installation of text-generation-webui.

Finally, press Clone. After the operation is finished you may close GitHub Desktop.

Initial setup

Installing text-generation-webui is straightforward. Use your file explorer to navigate to the text-generation-webui instalation folder you previously selected.

Now, find and double-click on a file named start_windows.bat (or, if you're using linux or macOS, start_linux.sh or start_macos.sh, respectively). You will use the same file in the future to update and launch the program.

Caution!
This script will automatically download approximately 3GB of data to your computer. Be aware of this especially if you're on a metered connection.

A terminal window will launch, downloading some required dependencies. At some point, you will be asked to select your GPU vendor. Choose the desired option, then confirm with ENTER.

Next, follow the on-screen prompts. Be patient, as the script may take some time to complete. Do not worry if the installation seems to be stuck.

The installation process is complete once you see "Running on local URL:..." in your terminal window.

At this point, do not close the terminal window. Instead, take note of the local URL displayed. Using your web browser of choice, navigate to the URL given. You should see the main interface. Congratulations, you have successfully completed the installation process!
